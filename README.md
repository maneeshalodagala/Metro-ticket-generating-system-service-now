# 🚇 Metro Ticket Generation System

## 📌 Project Overview

The Metro Ticket Generation System is a ServiceNow-based application that automates the process of booking metro tickets through the Service Portal. Users can enter journey details, submit a booking request, and receive a digital metro ticket. The system uses Flow Designer to automate record creation and stores all booking information in a custom Metro Ticket table.

## 📋 Project Workflow

1. Create a Custom Metro Ticket Table.
2. Create the **Book a Metro Ticket** catalog item.
3. Add Catalog Variables.
4. Configure Catalog Client Scripts for fare calculation.
5. Create a Flow Designer workflow.
6. Retrieve Catalog Variables.
7. Create a record in the Metro Ticket table.
8. Generate a QR Code for the booked ticket.
9. Display the QR Code to the user.
10. Store booking details for future reference.
